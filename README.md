# ⏳ D-Day 계산기

간단한 기념일 및 D-Day 계산기 웹 애플리케이션입니다.
목표 날짜까지 며칠이 남았는지, 혹은 특정 날짜로부터 며칠이 지났는지 쉽게 계산하고 목록으로 관리할 수 있습니다.

🔗 **[지금 바로 사용해보기 (Live Demo)](https://jeju-mandarin.github.io/d-day)**

---

## ✨ 주요 기능

* **D-Day / D+ 계산:**
    * 미래의 특정 날짜까지 남은 날짜 (D-)를 계산합니다.
    * 과거의 특정 날짜로부터 지난 날짜 (D+)를 계산합니다.
    * 당일은 'D-Day'로 표시됩니다.
* **기념일 제목 지정:** "생일", "시험일"처럼 각 항목에 원하는 제목을 붙일 수 있습니다.
* **목록 관리:** 계산한 D-Day가 목록으로 누적되어 여러 개의 기념일을 한눈에 볼 수 있습니다.
* **크로스 플랫폼 UI:**
    * PC와 모바일 모든 환경에서 동일하게 동작하는 반응형 웹 디자인을 적용했습니다.
    * **[flatpickr.js](https://flatpickr.js.org/)** 라이브러리를 사용하여, 모바일 기기에서도 직관적인 팝업 달력 UI를 제공합니다.
